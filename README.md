### Overview
A comprehensive, interactive web application exploring the **Iranian Constitution of 1906 (Mashruteh)**. This app features the full text of the constitution and its amendments, historical analysis, detailed timelines, and an AI-powered assistant (using Google Gemini) to answer legal and historical questions.

### Features
*   **Full Text Search:** Search through articles, amendments, and historical documents.
*   **AI Assistant:** Ask questions about the laws and history, powered by Google Gemini.
*   **Audio Narration:** Text-to-Speech functionality for articles and AI responses.
*   **Historical Timeline:** Visual timeline of events leading to and following the revolution.
*   **Trial Transcripts:** Detailed records of Dr. Mossadegh's trial.
*   **Theming:** Multiple visual themes (Modern, Dark, Royal, Cyberpunk, etc.) and font customization.
*   **Export Tools:** Export chats and documents to PDF, Word, HTML, and more.

### Quick Start
1.  Clone the repository.
2.  Run `npm install`.
3.  Create a `.env` file and add your API key: `API_KEY=your_google_gemini_key`.
4.  Run `npm run dev`.
